### 🎯 What it does
This project uses **MediaPipe** and **OpenCV** to detect and draw **468 facial landmarks** on your face in real-time through your webcam. No GPU needed — runs smoothly on CPU!

### 💡 Use Cases
- 🎭 Face filters and AR effects
- 💄 Virtual makeup try-on
- 🎬 Motion capture
- 😊 Emotion detection base
- 🕹️ Gesture-based control
